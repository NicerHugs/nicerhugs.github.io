My work

Dance Card

view code view site

Dance Card is a platform that connects dancers with dance organizers, finally making it easy to find dances anywhere in the country with a simple search. Dancers can find events based on location, distance, date, and type, and RSVP and keep track of events using their dance card.

Dance Card is also event management software for dance organizers, simplifying the hosting process, and ensuring that dances reach their maximum potential. Organizers can manage event details easily with built in recurring event management, and keep attendees up to date about changes with automated email notifications.

  • Parse/Backbone, Parse Cloud Code
  • Google Maps API, Mandrill API
  • jQuery, Underscore, Moment JS
  • HTML5, Sass, UI/UX design and implementation

This mock-up was created as a weekend homework assignment for The Iron Yard. The repositories tab is a complete, functional recreation of my Github repositories tab, kept up to date with AJAX calls to the public Github API.
